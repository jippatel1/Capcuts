How to Download CapCut and Start a New Project

Learn how to download CapCut and create your first editing project. Completing this task will prepare you to begin editing videos for social media.

Before starting , make sure you have an iPhone or iPad with internet access and enough storage space to install applications.
Before editing videos in CapCut, users must first download the application and create a project. Creating a project allows users to import video clips and begin organizing footage for editing.
  1. Open the App Store on your iPhone or iPad.
    The App Store icon is usually located on the home screen.
    The App Store home page appears.
  2. Search for "CapCut" using the search bar.
    Make sure to select the official CapCut application .
  3. Tap Get or Download to install the application.
    The app may require Face ID, Touch ID, or an Apple ID password before downloading.
    CapCut downloads and appears on your device home screen.
  4. Open the CapCut application.
    When opening the app for the first time, CapCut may request access to photos and videos.
    The CapCut home screen appears.
  5. Tap New Project.
    This option starts a new editing project and allows you to select media files.
    The photo and video library opens.
  6. Select the video clips or photos you want to edit.
    Choose clips in the order you want them to appear in your video.
    • • Select videos for short-form content.
    • • Select photos if creating a slideshow-style video.
    Selected clips are highlighted.
  7. Tap Add.
    Your clips appear in the editing timeline and the project is ready for editing.
You have successfully downloaded CapCut and created a new editing project. Your media is now ready to be arranged and edited.
Continue to How to Import, Arrange, and Trim Video Clips to begin organizing your footage.
